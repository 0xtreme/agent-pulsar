Step 4: Test & Connect

Everything is running. Let's connect Telegram.

1
Get a Telegram bot token
Open Telegram, search for @BotFather, send /newbot, and follow the prompts. Copy the bot token you receive.
2
Connect your bot
Paste your bot token below and click Connect. This automatically configures OpenClaw to use your Telegram bot.
3
Send a message
Open Telegram, find your bot, and send: "Research the latest trends in AI agents"

Or test via command line

Submit a task directly to the Supervisor API.

curl -X POST http://localhost:8100/tasks \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "you",
    "conversation_id": "test",
    "intent": "research.summarize",
    "raw_message": "What is event-driven architecture?",
    "params": {},
    "priority": "normal"
  }'

Check status with: curl http://localhost:8100/tasks/<request_id>
