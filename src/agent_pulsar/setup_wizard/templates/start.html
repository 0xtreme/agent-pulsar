Step 3: Start Services

Start Agent Pulsar with a single command.

Open a terminal in your Agent Pulsar directory and run:

./scripts/start.sh

This starts:

●
Redis
Event bus and cache (port 6379)
●
PostgreSQL
Task state database (port 5432)
●
Supervisor
Task decomposition and routing (port 8100)
●
Email Worker
Drafts professional emails
●
Research Worker
Researches topics, produces summaries
●
General Worker
Handles any other task type dynamically

To check status: ./scripts/start.sh status

To stop: ./scripts/start.sh stop
